Diven.t

O Driven.t foi feito para ser um site de venda de ingressos de todos os tipos, como por exemplo ingresso para eventos, sinemas e etc. Nele você pode fazer a reserva do seu ingresso para depois fazer a compra efetivamente, pode também escolher um hotel caso o ingresso escolhido tenha essa modalidade e por fim pode garantir seu lugar em qualquer palestra do evento.

Como foi feito ?

Esse é um projeto Full Stack. No front foi utilizadas as tecnologias HTML5, CSS3, ReactJS, StyledComponents, react-icons para fazer a inserção de icones na página, React-rout-Dom para fazer o roteamento do site, axios para lidar com requisições feitas ao nosso back-end, dayjs para lidar com as datas do site e etc..

O back-end foi escrito com TypeScript e foi utilizado o bcrypt para encriptar a senha do usuário no banco de dados, utilizamos express para subir o servidor e lidar com as rotas da nossa aplicação, dotenv para guardar as chaves secretas que não serão publicadas no github, foi usado o joi também para fazer a validação em tempo de execução das informações do usuário vindas do front, jsonwebtoken para gerar o token do usuário para que ele permanecesse logado no front e que conseguissemos autenticalo sem que ele precisasse ficar enviando email e senha toda hora, e etc..

No banco de dados optamos por utilizar o postgres para armazenar as informações do usuário, utilizamos o redis para que nossa aplicação tivesse uma scalabilidade maior também.

Nesse projeto foram feitos testes de integração para garantirmos que nossa api estivesse funcionando corretamente e para isso utilizamos a biblioteca jest.

Shortly

O shortly é um site voltado para para fazer um encurtador de url, para pessoas que estão com uma url grande poder encurtar sua url, funciona assim o usuário insere a url que quer encurtar o site pega essa url armazena e devolve uma url menor, sempre que o usuário digita a url no navegador que foi recebida ele acessa o site do shortly que é redirecionado para o site com o qual ele inseriu a url.

Cineflex

O cineflex foi desenvolvido com o pensamento de criar um site para venda de ingressos, ao contrário do Drivent ele é exclusivo para ingressos de cinema, onde o cliente pode escolher o filme, escolher um horário, por último escolher um assento disponível e feito isso é só digitar seu nome e cpf e comprar seu ingresso. há vale lembrar que o usuário pode escolher mais de um assento com o mesmo cpf.

Como foi feito ?

Esse é um projeto Front-End, e foi utilizadas as tecnologias HTML5, CSS3, ReactJS, React-rout-Dom para fazer o roteamento do site, axios para lidar com requisições feitas ao nosso back-end, e foi utilizado também a biblioteca animate.css para fazer as animações do site.
